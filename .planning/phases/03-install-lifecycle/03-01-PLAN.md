```sh
#!/bin/sh
# postinst for igh-seeedstudio
set -e

case "$1" in
    configure)
        # --- INST-01: Install blacklist for stock Realtek drivers ---
        # Use "install /bin/true" pattern to prevent udev bypass
        cat > /etc/modprobe.d/blacklist-eth.conf <<'BLACKLIST'
# Prevent stock Realtek drivers from loading.
# EtherCAT uses ec_r8169 instead.
install r8169 /bin/true
install r8168 /bin/true
BLACKLIST

        # --- INST-03 + INST-04: Detect MAC and write ethercat.conf ---
        IFACE="enP8p1s0"
        SYSFS_PATH="/sys/class/net/${IFACE}/address"

        if [ -f "${SYSFS_PATH}" ]; then
            MAC=$(cat "${SYSFS_PATH}")
        else
            echo "WARNING: ${SYSFS_PATH} not found, cannot detect MAC address" >&2
            echo "ethercat.conf will need manual configuration" >&2
            MAC=""
        fi

        cat > /etc/ethercat.conf <<CONF
# EtherCAT Master configuration
# Generated by igh-seeedstudio postinst
MASTER0_DEVICE="${MAC}"
DEVICE_MODULES="r8169"
CONF

        # --- INST-02 + service enable handled by #DEBHELPER# ---
        # dh_installmodules generates: depmod -a
        # dh_installsystemd generates: systemctl enable ethercat.service
        #DEBHELPER#

        # --- INST-05: Start service AFTER depmod (from #DEBHELPER# above) ---
        if [ -d /run/systemd/system ]; then
            systemctl restart ethercat.service || true
        fi
        ;;

    abort-upgrade|abort-remove|abort-deconfigure)
        #DEBHELPER#
        ;;

    *)
        echo "postinst called with unknown argument: $1" >&2
        exit 1
        ;;
esac

exit 0
```

Key design decisions reflected in this script:
1. Blacklist uses "install r8169 /bin/true" pattern (locked decision, not "blacklist r8169")
2. Also blacklists r8168 the same way (locked decision)
3. File written to /etc/modprobe.d/blacklist-eth.conf (locked decision)
4. depmod runs via #DEBHELPER# expansion from dh_installmodules (not hand-rolled)
5. Service start comes AFTER #DEBHELPER# (after depmod, locked ordering decision)
6. MAC detection from /sys/class/net/enP8p1s0/address with graceful fallback (hardcoded interface per PROJECT.md)
7. ethercat.conf uses shell variable format (IgH ethercatctl sources it with . operator)
8. case statement handles configure + abort-* cases (Debian Policy standard)
9. systemctl restart guarded by /run/systemd/system check (safe in Docker/chroot)
10. systemctl restart uses || true to not fail postinst if service fails to start (allows dpkg to complete)

DO NOT include update-initramfs -u (deferred to v2 per CONTEXT.md).
DO NOT include postrm purge logic (deferred to v2 per CONTEXT.md).

After writing the file, make it executable: chmod 755 debian/postinst
