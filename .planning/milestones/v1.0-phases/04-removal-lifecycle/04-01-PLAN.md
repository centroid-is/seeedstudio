Create the debian/prerm maintainer script that handles clean package removal by stopping the ethercat service and unloading EtherCAT kernel modules before dpkg removes files.

Purpose: Without prerm, dpkg -r would attempt to remove .ko files while modules are loaded and the service is running, causing removal errors and leaving state that prevents reinstall.
Output: debian/prerm executable maintainer script following the same POSIX sh patterns as debian/postinst.
</objective>

set -e

    configure)

        #DEBHELPER#

            systemctl restart ethercat.service || true
        fi
        ;;
    abort-upgrade|abort-remove|abort-deconfigure)
        #DEBHELPER#
        ;;
    *)

        exit 1
        ;;
esac
exit 0
